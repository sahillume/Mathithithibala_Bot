## ⚙️ Setup Guide

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/Mathithibala_Bot.git
cd Mathithibala_Bot
```

---

### 2️⃣ Install Dependencies

```bash
npm install
```

---

### 3️⃣ Configure Bot

Edit `config.js`:

```js
module.exports = {
    botName: "Mathithibala_Bot",
    ownerName: "Professor Sahil",

    ownerNumbers: [
        "27835515085",
        "27724469823"
    ],

    prefix: "!",
    sessionID: ""
};
```

---

### 4️⃣ Run Bot

```bash
node index.js
```

* Scan QR if no session
* Auto login if session added
